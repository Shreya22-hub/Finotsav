# 🎉 Finotsav

**Finotsav** is a smart event planning platform designed to make celebrations stress-free by combining **event management** with **budget and financial guidance**.

---

## 🚀 Features

- 🎯 Event Planning (Weddings, Birthdays, Parties, etc.)
- 💰 Budget Calculator for event cost estimation
- 📊 Financial guidance for better expense planning
- 🤝 Vendor coordination support
- 🌐 Multilingual support (via Google Translate)
- 💬 Chat support integration
- 📱 Fully responsive design

---

## 💡 Problem Statement

Planning events is stressful, especially for middle-class families.  
Managing vendors, budgets, and financial arrangements often leads to confusion and overspending.

---

## ✅ Solution

Finotsav provides an **all-in-one platform** that helps users:

- Plan events easily  
- Estimate budgets using a calculator  
- Get financial guidance  
- Coordinate vendors efficiently
